# Menu Flipbook (ฟรี 100%)
วิธีใช้งานแบบเร็ว:
1) ใส่ไฟล์เมนูของคุณเป็น PDF ที่ `assets/menu.pdf` (ตั้งชื่อนี้ให้ตรง)
2) อัปโหลดทั้งโฟลเดอร์นี้ขึ้น GitHub Pages / โฮสติ้งของคุณ / หรือเปิดไฟล์ `index.html` ตรง ๆ ก็ได้
3) เปิด `index.html` แล้วพลิกหน้าได้ทันที

ไฟล์ที่ใช้ CDN:
- jQuery
- Turn.js
- PDF.js

หากต้องการใช้งานแบบออฟไลน์สนิท ให้ดาวน์โหลดไฟล์ไลบรารีข้างต้นมาไว้ในโปรเจกต์แล้วแก้ลิงก์ให้ชี้เป็นไฟล์ภายในแทน
